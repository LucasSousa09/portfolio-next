# Portfolio v2.0.0

- A simply styled portfolio designed by me, inspired by design course portfolio videos.

![image](https://github.com/LucasSousa09/portfolio-next/blob/main/src/assets/portfolio-home-light.png)

- It also comes with a beautiful dark theme.

![image](https://github.com/LucasSousa09/portfolio-next/blob/main/src/assets/portfolio-home-dark.png)

## Technologies and Tools

- NextJS
- Tailwind CSS
- Phosphor Icons

## Running the app

- Download the application files
- Install the dependencies with: 'npm install' or 'yarn add'
- Run the application with: 'npm run dev' or 'yarn dev'
